Upkeep planner is a tool to keep track of recurring maintenance tasks. Organize tasks into categories, set their due dates, and store a record of past completions.

Each task has a different color circle depending on its status:

Due dates are highly customizable. Configure upcoming and grace periods to give yourself some wiggle room. Choose a set of active months for seasonal tasks.

Upkeep planner is an open-source application. Visit this repository to see the code.
